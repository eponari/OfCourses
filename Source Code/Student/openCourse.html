Computer Graphics Design.

Professor Vivian

Students enrolled: 90


Introduction to two-dimensional and three-dimensional graphics along with image processing.

Course Description:

Computer Graphics Design combines technology and art by using computer generated images also known as CGI and text to create concepts for publishing and advertising which portray different kinds of messages. Computer graphic designers, editors and animators use technology to manipulate and combine visual elements such as images with sound effects and words to illustrate emotions and deliver them clearly to an audience.

Key Skills: Artistic ability, creativity, analytical skills, communication skills, leadership, time-efficiency, computer skills.

Course Objectives:

This course introduces the basic concepts of Computer Graphics Design by providing the necessary theoretical material as well as enough practical applications connected with appropriate software programs which will make the workflow process smoothly. Further on, this course aims to teach students the programming languages needed for the program along with demonstrative scripts and assignments which can be solved both creatively and experimentally.

Weeks 14: 14

Lessons:

  • Week 1: Introduction to Computer Graphics Design.
  • Week 2: History of computer graphics and its evolution.
  • Week 3: The application of images, colors, sound effects and words.
  • Week 4: Programming languages. Homework 1.
  • Week 5: Laboratory programming practice.
  • Week 6: Most commonly known computer graphics software programs.
  • Week 7: Implementation of scripts into software programs. Homework 2.
  • Week 8: Create your first computer graphic object using C++.
  • Week 9: Create your first computer graphic object using Java.
  • Week 10: Quiz 1.
  • Week 11: The impact of computer graphics in different industries.
  • Week 12: How to add visual and audio elements to your graphical scripts.
  • Week 13: Rendering and exporting computer graphic designs. Final Project.
  • Week 14: The importance of computer graphics in the real world.
